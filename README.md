## 🚀 Getting Started

### Prerequisites

- Flutter SDK (3.x or higher)
- Android Studio / VS Code
- Android Emulator or physical device

### Installation

1. Clone the repository

git clone https://github.com/yourusername/quiz_app.git

2. Navigate to the project folder

cd quiz_app

3. Get dependencies

flutter pub get

4. Run the app

flutter run

## 📁 Project Structure
